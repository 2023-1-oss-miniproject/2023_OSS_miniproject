# 2023_OSS_miniproject

Mini-Project 주제 : 
  - 카페의 디저트 메뉴판 프로그램

Mini-Project에 대한 소개 : 
  - 카페의 디저트 메뉴들을 종류에 맞춰서 사용자에게 보여주는 프로그램입니다.
  - 메뉴가 추가 되면 메뉴의 종
  - 원하는 메뉴들과 수량을 입력하면, 가격을 알려줍니다.

Image : 

![bread-ge037cf06f_1920](https://user-images.githubusercontent.com/130240820/236674321-02963e9e-b8a9-487d-b647-8279d9a2cb27.jpg)

Project에 포함시킬 기능 설명 : 
  1. 메뉴판에 메뉴 추가 기능
  2. 메뉴판 출력 기능
  3. 메뉴판 수정 기능
  4. 메뉴를 메뉴판에서 삭제 기능
  5. 생성한 메뉴판 저장 기능
  6. 기존의 메뉴판 불러오기 기능
  7. 검색 기능
  8. 가격 계산 기능

개발 환경 및 언어 : 
  1. vscode
  2. C
  3. git

팀소개 및 팀원이 맡은 역할 : 
  - 한상화
	1. Repo Owner
	2. CRUD 구현
	3. wiki 제작

  - 이희준
	1. Contributer
	2. 코드관리 및 부가기능 구현
	3. wiki 제작
